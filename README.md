# ESP32 マルチ機能 ダッシュボード (顔アニメーション付き)

ESP32を使用した、滑らかな画面遷移と顔アニメーションを搭載した多機能情報端末です。
室内の温湿度、地域の天気、ニュース、Minecraftサーバーの稼働状況などをリアルタイムに表示します。

## 🚀 主な機能

- **動的な表情 (Face Mode)**: 視線移動やまばたきを行うインタラクティブな表情。通知受信時には表情が変化します。
- **リアルタイム情報表示**:
    - **時計**: NTPサーバーと同期した正確な時刻。
    - **天気**: OpenWeatherMap APIを利用した外気温と天候の表示。
    - **室内環境**: DHT11センサーによる温度・湿度と、直近の温度変化グラフ。
    - **ニュース**: NHK RSSフィードから最新ニュースをスクロール表示（自動ジャンル切り替え）。
    - **Minecraft**: 指定したサーバーの稼働人数とMOTDを表示。
- **Webダッシュボード**: ブラウザから室温グラフの確認、設定（都市名・サーバー名）の変更、通知の送信が可能。
- **デュアルコア最適化**: 通信処理と描画処理を別々のコアで実行し、カクつきのない動作を実現。

## 🛠 ハードウェア構成

- **マイコン**: ESP32 (DevKit V1など)
- **ディスプレイ**: 0.96インチ OLED (SSD1306 / 128x64 / I2C)
- **センサー**: DHT11 温湿度センサー
- **操作**: タクトスイッチ x 3 (左、右、決定/固定)
- **通知**: 内蔵LED (GPIO 2)

### 配線図


| パーツ | ピン | ESP32 |
| :--- | :--- | :--- |
| OLED | SCL / SDA | GPIO 22 / 21 |
| DHT11 | DATA | GPIO 4 |
| Button L | - | GPIO 12 |
| Button R | - | GPIO 13 |
| Button Sel| - | GPIO 14 |

## 💻 ソフトウェアの準備

### 必要なライブラリ
Arduino IDEのライブラリマネージャーから以下をインストールしてください：
- `U8g2` (OLED描画)
- `DHT sensor library` (センサー)
- `Adafruit Unified Sensor` (センサー共通)
- `ArduinoJson` (JSON解析)
